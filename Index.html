Thema 01 / 03

Kondensator & Spule

Verhalten an Gleichstrom – qualitativ, ohne Rechnen. Lade-/Entlade-Kurven, Kapazität, Induktivität.

Spannung U Strom I
Kondensator Laden:
Beim Einschalten fließt der Strom sofort maximal, dann sinkt er exponentiell. Die Spannung steigt langsam auf den Endwert.

Wenn voll geladen: kein Strom mehr → Kondensator blockiert Gleichstrom.

Merksatz: Am Anfang = Kurzschluss · Am Ende = offener Schalter
🔋 Kapazität C
Maß für die speicherbare Ladung bei einer bestimmten Spannung.

Einheit: Farad (F)
Größeres C → langsamere Lade-Kurve
🌀 Induktivität L
Maß für den Widerstand der Spule gegen Stromänderungen. Energie im Magnetfeld gespeichert.

Einheit: Henry (H)
Größeres L → langsamerer Stromanstieg
Thema 02 / 03

Lasten- & Pflichtenheft

Dokumente in der Softwareentwicklung – definieren WAS und WIE gebaut wird.

Auftraggeber
📋 Lastenheft
„Was wollen wir haben?"
↓
Auftragnehmer
📝 Pflichtenheft
„Wie setzen wir es um?"
↓
🚀 Fertiges Produkt
📋 Lastenheft
Was soll das System können?
Anforderungen des Auftraggebers
Grob, nicht technisch
Geschrieben vom Auftraggeber
Keine Lösungsdetails
📝 Pflichtenheft
Wie wird es technisch umgesetzt?
Technische Spezifikation
Detailliert & präzise
Geschrieben vom Auftragnehmer
Konkrete Lösungsbeschreibung
Konkretes Beispiel:
Lastenheft: „Wir brauchen eine App, mit der Schüler ihre Hausaufgaben verwalten können."

Pflichtenheft: „Die App wird in React Native entwickelt, hat eine SQLite-Datenbank, Push-Benachrichtigungen, Login via Google OAuth..."

Merksatz: Lastenheft = Laie schreibt  ·  Pflichtenheft = Profis antworten

    🔗 Zusammenspiel der Komponenten
    ⚙️ CPU ↔ RAMProgramme laufen im RAM, die CPU berechnet sie
    🔲 Mainboardverbindet alle Teile – wie das Nervensystem des PCs

    CPU = Gehirn  ·  RAM = Kurzzeitgedächtnis  ·  SSD = Langzeitgedächtnis
